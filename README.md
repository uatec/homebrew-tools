# Homebrew Tools Tap

A custom Homebrew tap for installing formulae and casks.

## Installation

```bash
brew tap uatec/tools
```

## Usage

### Installing Formulae

```bash
brew install uatec/tools/<formula>
```

### Installing Casks

```bash
brew install --cask uatec/tools/<cask>
```
